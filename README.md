Tortuga HOI
===========


## Hoi

Hoi skal **H**ente **O**pplysninger om **I**nntekt til personer og tilby disse på en Kafka-topic.

## Installasjon og kjøring

### Bygging

For å bygge JAR og tilhørende Docker images:

```
make
```

Man kan også bygge med maven:

```
mvn package
```
